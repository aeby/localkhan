localkhan

Download and distribute Khan Academy content to any devices. You can then use the Khan exercises and videos in a simple web-based client without a network connection.

Get started with 4 simple steps

1. Install localkhan

$ pip install localkhan

2. Download the desired Khan (sub-)topic

$ localkhan get early-math

You can identify the topic path by visiting the desired topic on www.khanacademy.org and get the path after the subject name from the URL like this:

  • https://www.khanacademy.org/math/early-math
  • https://www.khanacademy.org/math/arithmetic/multiplication-division

3. Start local server

$ localkhan serve

4. Visit your local server on your target device
